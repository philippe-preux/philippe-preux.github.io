Énoncé

Nous allons étudier un jeu de données qui contient toutes les éruptions volcaniques connues s'étant déroulées jusqu'en 2020.
Ce fichier est disponible à l'url https://philippe-preux.github.io/ensg/miashs/datasets/eok.csv.
Pour chaque éruption, le jeu de données indique :

  • un numéro du volcan concerné ;
  • le nom du volcan ;
  • un numéro d'éruption ;
  • le type d'éruption ;
  • la zone d'activité du volcan ;
  • l'intensité de l'éruption (Volcanic explosivity index) ;
  • la date de début d'éruption ;
  • la méthode utilisée pour dater l'éruption ;
  • la date de fin de l'éruption ;
  • la latitude du volcan ;
  • la longitude du volcan ;
  • le pays où est situé le volcan.

Les questions :

  1. quelle commande effectuez-vous pour charger le jeu de données. Faites-le ;
