Valeur de toutes les politiques d'un PDM

  1. Écrire une fonction qui calcule la fonction valeur de toutes les politiques existants dans un PDM.
  2. Appliquer cette fonction au problème du chauffeur de taxi et afficher toutes ces fonctions valeurs sous la forme d'un ensemble de lignes, chacune affichant la valeur des 3 états de ce PDM.
  3. Écrire une fonction qui détermine laquelle de ces fonctions valeurs est maximale.
  4. L'appliquer au problème du chauffeur de taxi.
  5. Essayer de reproduire la figure 3 du polycopié.

Vous m'envoyez votre code qui doit afficher ces fonctions valeurs les unes après les autres, une fonction valeur par ligne et une ligne par politique. On doit obtenir un affichage comme suit : 

 (0, 0, 0) : 91.25740330717487, 97.55101752705636, 89.9670807265297
 (0, 0, 1) : 91.94946449326041, 98.29080707438501, 90.91899632382766
 (0, 0, 2) : 83.4209262468556, 89.17409377383824, 79.18817167153884
 ...
 (2, 2, 2) : 74.93627214901949, 109.93397160609136, 73.70318113954124

Quand je lance votre code, il doit afficher cela (sans que je doive chercher comment il faut faire).

Politique optimale en fonction de γ

On a vu en cours que la politique optimale dépend de la valeur de γ. Plus précisément, on a vu que l'intervalle de valeurs de γ ∈ [0, 1[ est découpé en segments sur chacun desquels la politique optimale reste constante.

  • Écrire une fonction qui balaie les valeurs de γ ∈ [0, 1[ et affiche la politique optimale sur chacun des segments. (Les bornes du segment peuvent être approximatives.)
  • Appliquer votre fonction au problème du chauffeur de taxi. Vous découpez l'intervalle [0, 1[ en n valeurs. Prenez n = 1000.
    Par exemple, si la politique optimale change pour γ = 0.1, 0.23, 0.43, 0.45 0.78, 0.9 (ce ne sont pas les bonnes valeurs, c'est juste pour donner un exemple), on obtient un affichage comme celui-ci : 
          0.0 : les actions optimales pour chacun des 3 états 
      0.1 : les actions optimales pour chacun des 3 états pour γ ∈ [0, 0.1[
      0.23 : les actions optimales pour chacun des 3 états pour γ ∈ [0.0.23, 0.43[
      0.43 : les actions optimales pour chacun des 3 états pour γ ∈ [0.43, 0.45[
      0.45 : les actions optimales pour chacun des 3 états pour γ ∈ [0.45, 0.78[
      0.78 :les actions optimales pour chacun des 3 états pour γ ∈ [0.78, 0.9[
      0.9 : les actions optimales pour chacun des 3 états pour γ ∈ [0.9, 1[

Quand je lance votre code, il doit afficher cela (sans que je doive chercher comment il faut faire).
