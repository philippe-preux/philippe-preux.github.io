Pour accéder aux exemples à utiliser durant le contrôle, insérez les commandes suivant dans votre programme python : 

from numpy import genfromtxt
olives = genfromtxt ("https://philippe-preux.github.io/ensg/l1-mi/tps/cc2/olives3.csv", delimiter = ',')

entrées_olives = olives [:,[0,1]]
sorties_olives = olives [:,2]

sorties = []
for i in range (len (sorties_olives)):
    if sorties_olives [i] == 1:
        sorties.append (0)
    else:
        sorties.append (1)

Ces instructions produisent les entrées et les sorties correspondantes, exactement comme nous l'avons utilisé avec les iris.
