Objectifs du TP

ggobi est capable d'effectuer une ACP. Comme on l'a vu en cours, une ACP n'est rien d'autres qu'une projection particulière du jeu de données. Celle-ci a la propriété d'être effectuée dans un repère orthogonal dans lequel les axes sont ordonnés et l'allongement (la variance) du jeu de données selon chacun des axes diminue en suivant l'ordre sur les axes. On va donc voir ici comment calculer l'ACP et la représenter dans ggobi.

Faire une ACP revient à décorréler les données de manière à ce que la matrice de covariance soit diagonale. Cette opération porte aussi le joli nom de sphering en anglais et est nommée ainsi dans ggobi. On pourra consulter les informations sur ce lien concernant le sphering ainsi que mes notes de cours sur le sujet.

Dans le menu « Tools », sélectionnez « Sphering ... ». Une fenêtre s'ouvre.

En haut de cette fenêtre, vous trouvez la liste des attributs. Sélectionnez-les tous.

Dessous, vous trouvez un bouton indiquant « Use correlation matrix ». Si vous ne cliquez pas sur ce bouton, les calculs sont faits en utilisant la matrice de variance. Si vous cliquez dessus, ils sont faits en utilisant la matrice de corrélation.
Pour bien voir la différence, faire le petit calcul suivant. Calculez la covariance puis le coefficient de corrélation entre les attributs taille et poids pour les données ci-dessus. Maintenant, refaites ces deux calculs en considérant la taille en centimètres. Concluez.

En cliquant sur le bouton « Update scree plot », on déclenche le calcul de l'ACP et se dessine le spectre des valeurs propres.

Encore un clic un peu plus bas sur le bouton « Apply sphering », et les coordonnées des points sont calculées dans l'espace factoriel.

Vous constatez que des boutons ont été ajoutés dans la fenêtre de contrôle de ggobi. Ils sont étiquetés PC1, PC2, PC3 et PC4. PC signfie principal component. Vous avez donc maintenant tous les éléments en main pour représenter le jeu de données dans le plan principal et dans tous les plans de l'espace factoriel, voire d'y faire un grand tour.
