Mise en pratique

J'utilise R comme environnement logiciel support pour mes cours de fouille de données.

Ci-dessous, des pointeurs vers des sujets de TP abordant les différents points de mon cours :

  • découverte de R ; voir aussi généralités sur R
  • classification supervisée par arbres de décision
  • exercice de probabilités
  • segmentation de données (classification non supervisée) :
    • segmentation de données par les k-moyennes ;
    • segmentation hiérarchique ascendante de données ;
    • segmentation par les k-médianes ;
    • exploration visuelle des données
  • préparation des données
  • réduction de dimension (ACP, MDS)
  • visualisation de données avec gobi :
    • découverte de Gobi,
    • une ACP dans Gobi,
    • représentation d'un graphe,
    • analyse de dissmilarité.
    Gobi est ici utilisé indépendemment de R. Le paquet Rggobi permet maintenant d'utiliser directement Gobi depuis R.
